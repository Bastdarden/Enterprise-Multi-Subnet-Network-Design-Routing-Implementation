## Enterprise Multi-Subnet Network Design & Routing Implementation

### Project Overview

Designed and implemented a scalable enterprise network infrastructure for a simulated mid-sized organization using Cisco Packet Tracer.

The project focused on creating a segmented network architecture capable of supporting multiple departments while improving scalability, simplifying network management, and enabling secure communication across subnets.

Key technologies included:

-IPv4 Addressing
-Variable Length Subnetting
-DHCP
-Static IP Assignment
-Cisco IOS
-RIPv2 Dynamic Routing
-Network Validation
-Troubleshooting

### Business Scenario

TechLink Solutions required a scalable network capable of supporting future organizational growth while maintaining efficient communication between business departments.

Project objectives included:

Design a segmented network
Implement dynamic routing
Configure DHCP services
Assign static infrastructure addresses
Verify end-to-end connectivity
Produce implementation documentation
